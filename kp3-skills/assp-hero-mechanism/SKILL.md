# ASSP — Hero Mechanism Builder
## Agent 03/12 · ASSP Framework

---

## Purpose

Build a proprietary mechanism — a name and system for how you produce results — that makes customers unable to compare you to anyone else.

**Core logic:**
When you have your own mechanism, you're no longer selling "a baking course" — you're selling the "7-Layer Emotion Method". No one else has that. Price comparison becomes impossible.

Final output: **hero-mechanism.md** file — used in all copy, offers, and ads going forward.

---

## Before Starting

If **avatar.md** already exists in this chat → read and use it directly.

If not → ask quickly:
> "Who are you helping and what result do you help them achieve? Describe briefly."

---

## Process — 5 Steps

**⚡ Read `references/mechanism-steps.md` before starting.**

That file has all the detailed questions, how to handle each step, and reference examples.

**Summary of 5 steps:**

```
Step 1: Understand how you produce results
         → Ask about process, steps, what's genuinely different
         → Don't accept vague answers

Step 2: Verify "can't be compared"
         → Research competitors yourself
         → Identify what's truly different — not just "better quality"

Step 3: Name the Mechanism
         → Suggest 4-5 names from different angles
         → Analyze each + identify the best one
         → See references/mechanism-examples.md for reference

Step 4: Build the complete mechanism
         → Name + number of steps/phases + description of each part
         → Check: does a customer understand it immediately after hearing it?

Step 5: Guide how to use the mechanism throughout
         → How to embed it in the offer, ads, VSL, email
```

## Handling Difficult Cases

**"I do the same as everyone else, nothing special":**
> No one does exactly the same thing. Dig deeper: Is your sequence different? What do you focus on most that others skip? What's distinctive about your clients' results?

**"The name sounds good but I'm not sure":**
> Test 3 questions: (1) Does hearing it make someone curious to know more? (2) Can you explain it in 10 seconds? (3) Could a competitor use this name? If (3) is "yes" → the name isn't distinct enough.
