# ASSP — Sales Call Script
## Agent 12/12 · ASSP Framework

---

## Purpose

Write a complete closing call script — from the opening line to collecting payment or ending gracefully.

**Philosophy:** Sell like a doctor — diagnose the problem first, prescribe the solution second.
The caller booked a call because they have a problem to solve — not to hear you advertise.

---

## Before Starting

**If context from previous agents already exists in this chat → read and use directly.**

If not → ask:
> "Paste your **offer-[name].md** here if you have it."

Then ask 3 questions:

> **Q1 — What is the call's goal?**
> *Qualify-only vs. qualify-and-close affects the entire script structure.*
>
> **A. Qualify + close on the call**
> → Right when: the offer is clear, the caller has already seen the sales page and filled out the form
> → Script needs all 6 phases including the close and objection handling
>
> **B. Qualify only — close via payment link after**
> → Right when: you want to separate qualifying from selling, or the offer needs more thought
> → Script only needs phases 1-4, ending with sending the link

> **Q2 — Who will be making the calls?**
> *Different callers need different script tones.*
>
> **A. You make the calls yourself** → Natural script in your voice, can be flexible
> **B. A staff member makes calls** → More detailed script, less flexibility, needs specific examples

> **Q3 — Which objections come up most often?**
> *The script needs to handle the real objections — not generic ones.*
>
> **A. Price** → Need strong ROI justification section
> **B. Time** → Need section explaining the real time commitment
> **C. "Let me think about it"** → Need deep-dive and test close section
> **D. Need to ask someone else** → Need follow-up appointment section
> **E. All of the above** → Write all 4

## Process

**⚡ Read `references/sales-call-framework.md` first — has a Table of Contents to find the right section.**

Write the script following the 6 phases in the framework.
Each phase includes:
- Goal of that phase
- Specific lines — natural Vietnamese, speakable
- Deep-dive questions if needed
- Signal to move to the next phase

---

## Required Script Format

```
## PHASE [number] — [Name] ([X minutes])
**Goal:** [1 sentence]
**Signal to move on:** [when to go to next phase]

---

**[Caller]:**
> "[Script line — natural Vietnamese, speakable out loud]"

**[Possible caller responses:]**
> "[Common responses]"

**[Caller — if response A:]**
> "[Next line]"

---
💡 **Why this step:** [Brief explanation of the logic]
```

---

## Script Writing Rules — Required

**All written in natural Vietnamese:**
Read it aloud — if it sounds stiff or like a translation → rewrite.

**Open-ended questions — no yes/no questions:**
```
❌ "Do you want to join?"
✅ "What's holding you back from starting?"
```

**Listen more than you talk:**
The script must have clear [silence and listen] moments.
The caller should be talking at least 60% of the time.

**Never speak badly about competitors.**

**After stating price — complete silence:**
This rule cannot be broken. Note it clearly in the script.

**Don't push people who aren't a fit:**
The script must include a "graceful ending" step for when the caller isn't the right fit.

---

## Output Language — Required

All outputs must be written in **Vietnamese** — simple, natural, easy to understand.
Do not use English marketing terms when a Vietnamese equivalent exists.
